NAME
       docker-plugin-push - Push a plugin to a registry

SYNOPSIS
       docker plugin push [OPTIONS] PLUGIN[:TAG]

DESCRIPTION
       Push a plugin to a registry

OPTIONS
       --disable-content-trust[=true]	   Skip image signing

       -h, --help[=false]      help for push
