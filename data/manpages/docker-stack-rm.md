NAME
       docker-stack-rm - Remove one or more stacks

SYNOPSIS
       docker stack rm [OPTIONS] STACK [STACK...]

DESCRIPTION
       Remove one or more stacks

OPTIONS
       -d, --detach[=true]	Do not wait for stack removal

       -h, --help[=false]      help for rm

SEE ALSO
       docker-stack(1)
