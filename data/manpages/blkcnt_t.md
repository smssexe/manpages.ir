NAME
       blkcnt_t - file block counts

LIBRARY
       Standard C library (libc)

SYNOPSIS
       #include <sys/types.h>

       typedef /* ... */ blkcnt_t;

DESCRIPTION
       Used for file block counts.  It is a signed integer type.

STANDARDS
       POSIX.1-2008.

HISTORY
       POSIX.1-2001.

NOTES
       The following header also provides this type: <sys/stat.h>.

SEE ALSO
       stat(3type)
